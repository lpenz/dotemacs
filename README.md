# dotemacs

My ~/.emacs.d directory

## Install instructions

Manually:

    $ git clone git@github.com:lpenz/dotemacs.git ~/.emacs.d


... or using ansible-pull:

    $ ansible-pull -U https://github.com/lpenz/dotemacs.git -i localhost, ansible-playbook.yml
